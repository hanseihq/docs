This guide provides step-by-step instructions for integrating Hansei with Google Docs using Zapier. By following these steps, you can automate processes between Docs and Hansei, enhancing efficiency and organization. 
 
## Prerequisites 
- A Zapier account 
- A Google account with administrative privileges 
- Access to Hansei 
 
### Step 1: Initiate Integration 
1. Log in to your Zapier account 
2. Navigate to the Dashboard 
3. Start the integration process with Google Docs. 
 
<img 
  src="/images/docs/1.png" 
  alt="Integrate Docs with Hansei" 
/> 
 
### Step 2: Choose a Trigger Event. Options Include: 
'New Document' & 'New Document in Folder'.  
In this case, we choose the 'New Document' event. 
 
<img 
  src="/images/docs/2.png" 
  alt="Choose Event for Trigger" 
/> 
 
### Step 3: Connect to Docs 
1. Click Sign in to Docs. 
2. Enter your Google credentials and authorize Zapier to access your Google Docs account. 
 
<img 
  src="/images/docs/3.png" 
  alt="Connect Account" 
/> 
 
### Step 4: Configure the Trigger 
1. Select the specific Docs you want to monitor. 
2. Click on Continue. 
3. Test the trigger to ensure it's working correctly. 
 
<img 
  src="/images/docs/4.png" 
  alt="Test Trigger" 
/> 

### Step 5: 
Choose Hansei from the Action Menu 
 
<img 
  src="/images/docs/5.png" 
  alt="Setup Action" 
/> 

### Step 6: Set Up the Action 
1. In the 'Choose App & Event' section, search for Hansei and select it. 
2. Choose an action event from the following options: 
   - Add File to Knowledge Base 
   - Add Text Content to the Knowledge Base 
   - Add Webpage Content to Knowledge Base 
   - Generate Answer 
 
For this guide, select Add Text Content to Knowledge Base. 
 
<img 
  src="/images/docs/6.png" 
  alt="Setup Action" 
/> 

### Step 7: Configure the Action 
1. Fill in the details for the action, including: 
   - Select Collection: Choose the relevant collection in Hansei. 
   - Filename 
   - File  
2. Click on Continue. 
3. Test the action to ensure it's configured correctly. 
 
<img 
  src="/images/docs/7.png" 
  alt="Required fields in Action" 
/> 
 
<img 
  src="/images/docs/8.png" 
  alt="Complete all Fields" 
/> 
 
<img

You have successfully integrated Hansei with Google Docs using Zapier. This integration will streamline the process of adding content from Docs to your Hansei Knowledge Base, facilitating better knowledge management and collaboration.
